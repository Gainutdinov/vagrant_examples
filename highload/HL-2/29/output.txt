Задание:
Задача подразумевает, что у вас уже есть работающий EFK-кластер (см. задачу HL 33 «EFK. Fluent. Единичный экземпляр») и он собирает логи от тестового контейнера web и тестовой виртуальной машины (см. задачу HL 34 «EFK. Fluent. Настройка клиентского агента»).
1. Откройте Kibana в браузере, перейдите на страницу Discover и найдите поле для ввода Search.
2. Раскройте одно из сообщений. Изучите поля записи в табличном представлении и ознакомьтесь с JSON. Если нужно, измените диапазон дат.
3. Напишите запрос для поиска логов от данного хоста.
4. Напишите запрос для поиска логов от всех хостов, кроме данного хоста.

```
NOT (host:"coredns-slave")
```

5. Напишите запрос для поиска логов от данного контейнера.

```
host:"coredns-slave"
```

6. Напишите запрос для поиска логов от всех хостов, имя которых начинается с данного паттерна.

```
host:"coredns*"
```

7. Напишите запрос для поиска логов от двух данных хостов (2 варианта).

```
host:"coredns1" AND host:"coredns2"
```

8. Напишите запрос для поиска логов от сервиса SSH данного хоста.

```
"ident": "sshd"
```

9. Сохраните запрос п.8.
10. Воспользуйтесь ранее сохраненным запросом.

```
1.png
```
