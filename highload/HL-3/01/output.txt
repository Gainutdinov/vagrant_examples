Задание:
1. Для выполнения задания необходимо создать такую инфраструктуру:
○ ВМ1 mariadb со следующими параметрами: 1 CPU / 2G memory;
○ ВМ2 wordpress со следующими параметрами: 1 CPU / 1G memory;
○ ВМ3 jmeter со следующими параметрами: 1 CPU / 2G memory.
2. Примечание: возможно, потребуется обновить систему на ВМ[1-3].
3. На ВМ1 mariadb:
○ установите мета-пакет mariadb-server;
○ включите привязку MariaDB к внешнему IP-адресу ВМ1;
○ рестартуйте сервис mariadb-server;
○ подключитесь к MariaDB серверу пользователем root с пустым паролем.
4. Используя открытое подключение:
● создайте пользователя БД wordpress@<ВМ2_ip_адрес> с паролем wordpress;
● создайте БД wordpress;
● задайте пользователю wordpress полные права на БД wordpress.
Подключитесь к серверу БД пользователем wordpress.
5. На ВМ2 wordpress:
○ установите мета-пакет wordpress и запустите портал через веб-сервер apache (он уже установлен в рамках мета-пакета);
○ cоздайте файл конфигурации Wordpress /etc/wordpress/config-<ВМ2_ip_адрес>.php и сконфигурируйте подключение к БД на сервере mariadb через ранее созданного пользователя.
6. Откройте в браузере URL Wordpress http://<ВМ2_ip_адрес>/blog, введите данные сайта, авторизуйтесь.
7. В Wordpress установите плагин для basic authentication.

```
plugin name: WordPress REST API Authentication
```

8. На ВМ3 jmeter установите Apache Jmeter 5.2.
9. В JMeter GUI:
○ откройте созданный ранее тест-план TestPlan_main_page_1.jmx;
○ измените IP-адрес тестируемого сайта;
○ измените путь на тестируемом сайте /blog;
○ добавьте HTTP Authorization Manager для обеспечения авторизации;
○ добавьте HTTP Request для создания поста;
○ добавьте HTTP Header Manager для управления заголовками;
○ установите значения Number of Threads и Loops Count в 1;
○ убедитесь, что тест создает посты в Wordpress.
10. На ВМ3 jmeter запускайте тест с созданным тест-планом, увеличивая значение Number of Threads и Loops Count до тех пор, пока время отклика Average не превысит 3с. Зафиксируйте полученные значения.

```
Number of Threads (users): 35
Loop Count: 35
```
